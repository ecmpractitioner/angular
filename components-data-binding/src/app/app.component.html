• We can use alias name for binding the component. Ex: @Input('srvElement')
  • To pass data from child to parent component, use EventEmiiter
  • In the parent componenet, listen to the events from child and use the data
  • We can use local ref instead of two-way binding to get value of an HTML element.
    • When two-way binding [ngModel] is used, it creats FormsControl and we need to inport FormsModule
    • Use local ref when we just need a value or the entire HTML element
